## 经典搭配方案

### 方案 1: 北欧简约
```
主色: #FFFFFF (白色), #F5F5F5 (浅灰)
辅色: #2C3E50 (深蓝灰), #E8D5B7 (米色)
点缀: #D4A574 (陶土色)
```

### 方案 2: 法式浪漫
```
主色: #F8F4F0 (奶油白), #D4A5A5 (莫兰迪粉)
辅色: #9CAF88 (鼠尾草绿), #8FA3B0 (莫兰迪蓝)
点缀: #C8A96A (香槟金)
```

### 方案 3: 日式禅意
```
主色: #FFFFF0 (象牙白), #4A4A4A (炭灰)
辅色: #1B4965 (靛蓝), #8FBC8F (抹茶绿)
点缀: #CD853F (原木色)
```

### 方案 4: 现代奢华
```
主色: #1C1C1C (炭黑), #F5F5F5 (银白)
辅色: #B8860B (暗金), #800020 (酒红)
点缀: #C0C0C0 (银色)
```

### 方案 5: 波西米亚
```
主色: #E8D5B7 (米色), #8B4513 (赭石)
辅色: #008080 (湖水绿), #CC7722 (赭黄)
点缀: #FF7F50 (珊瑚), #800020 (酒红)
```

### 方案 6: 热带度假
```
主色: #FFFFFF (白色), #00CED1 (青绿)
辅色: #FF6B6B (珊瑚红), #FFD93D (阳光黄)
点缀: #4ECDC4 (薄荷), #FF69B4 (粉红)
```
